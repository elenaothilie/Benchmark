# PP Talisman font

Add PP Talisman font files here. Get them from [Pangram Pangram](https://pangrampangram.com/products/talisman).

Required files (rename to match):
- `PP-Talisman-Regular.woff2` (400)
- `PP-Talisman-Medium.woff2` (500)
- `PP-Talisman-SemiBold.woff2` (600)
- `PP-Talisman-Bold.woff2` (700)
- `PP-Talisman-ExtraBold.woff2` (800)

Or use a single variable font if available:
- `PP-Talisman-Variable.woff2`
